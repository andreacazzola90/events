# PWA Icons Setup

## Icone necessarie per la PWA

Devi creare le seguenti icone:

1. **icon-192x192.png** (192x192 pixels)
2. **icon-512x512.png** (512x512 pixels)

### Come crearle:

1. Usa un'immagine quadrata del tuo logo/icona
2. Ridimensionala a 192x192 e 512x512 pixels
3. Salvale come PNG nella cartella `/public`

### Strumenti online:

- https://realfavicongenerator.net/ (genera tutte le icone PWA)
- https://www.pwabuilder.com/imageGenerator (PWA Image Generator)
- Canva o Photoshop per creare icone personalizzate
